# Enhanced COW-QKD Simulation

This repository contains the Python code and datasets for the paper:
**"Enhancing the security of coherent one-way quantum key distribution using CHSH correlations"**

## Overview
This project simulates the proposed enhanced COW-QKD protocol. The code calculates the security bounds and verifies the performance based on CHSH inequality violations.

## Prerequisites
The simulation requires the following Python libraries:
- numpy
- matplotlib
- math

## Usage
1. Clone this repository: `git clone [https://github.com/alihasani0706/COW-QKD-simulation]`
2. Run the main simulation script:
   `python main_simulation.py`
3. The generated plots will be showed.
